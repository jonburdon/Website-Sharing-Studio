# Website Sharing Tools

A suite of tools for generating, validating, and sharing web links—starting with QR codes.

## Project Plan

### Phase 1 – MVP
- **URL to QR code generator** — Simple input, instant QR code
- **URL validator** — Validate URLs before generating
- **Download options** — PNG, SVG
- **Print-ready SVG** — High-quality vector for printing
- **Copy to clipboard** — Quick copy of generated QR image

### Phase 2 – Polish
- **Custom colors** — Foreground/background for branding
- **Size presets** — Small, medium, large for different use cases
- **PDF download** — Business card style printable layout
- **Short URL + QR combo** — Generate short link and its QR code together
- **Dark/light QR variants** — For different backgrounds

### Phase 3 – Accounts & Features
- **Log in and save** — Save generated QR codes to your account
- **Batch generation** — Generate multiple QR codes from a URL list
- **Recent history** — Last 5–10 generated (no login needed)
- **Error correction level** — For print durability

### Phase 4 – Extras
- **vCard QR codes** — Contact card QR codes
- **WiFi QR codes** — Share network name + password
- **Email QR codes** — Pre-filled mailto links
- **Social share buttons** — One-click share to Twitter, LinkedIn, etc.
